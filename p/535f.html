误差的类型

  • 误差是客观存在的且难以避免

系统误差

  • 系统误差由某个可确定的原因,按确定的规律引起
  • 有重复性,单向性,可测性和可避免性

解决方案: 在找出原因后可予以校正,如:

  • 方法误差: 换用合适恰当的方法
  • 仪器误差: 使用前记得进行校准
  • 试剂误差: 进行空白试验,将空白值从分析结果中扣除

随机误差

  • 由偶然因素引起,难以避免
  • 有不可测性和双向性
  • 出现概率符合正态分布,大量实验下误差平均值趋于0

解决方案: 一般平行测定3-4次,高要求则5-9次

过失误差

  • 由操作者操作不当引发,必须避免

解决方案: 删除所得测定结果

减小测量误差

  • 电子天平称量试剂的质量要大于 0.2 g
  • 滴定体积要大于 20 ml ,通常为 20~30 ml

相对误差 = (绝对误差 / 试样的量) * 100%

误差的表征与表示

误差与准确度

  • 准确度: 在一定条件下,测量值与真实值接近的程度(越高越接近).它的高低可用误差的大小来测定.

绝对误差(E) = 测量值 - 真实值
相对误差(RE) = 绝对误差(E) / 真实值
~ 相对误差一般用千分号表示

  • 被测物质总量越大的,在绝对误差相同时,相对误差就越小,误差对结果准确度影响也越小
  • 绝对误差和相对误差都有正负之分
  • 相对误差反映了误差在真实值中所占比例,更有实际价值

偏差与精密度

  • 精密度: 在相同情况下,多次平行测量时各测定结果相互接近的程度.它体现了测量结果的重现性,常用偏差来衡量

绝对偏差(偏差) = 个别测定值 - 算术平均值
相对偏差 = 绝对偏差 / 算术平均值

  • 平均值代表了测定值的集中趋势

  • 偏差则反映了测定值的分散程度,分散程度越小,精密度越高

  • 绝对偏差之和为零,则无法反映精密度.因此通常采用 平均偏差 和 相对平均偏差

平均偏差 = 各测量值偏差绝对值之和 / 测量值总数
相对平均偏差 = 平均偏差 / 测量平均值
二者均无负值

总结

  • 系统误差决定了测定结果的准确性,随机误差决定了结果的精密性
  • 一个优良的测量结果,既要有好的精密度,又要有高的准确度
  • 只有消除了系统误差,才能同时实现高的精密度和准确度
